# 🚐 Projeto Integrador - Gestão Inteligente de Transporte Escolar
Este projeto visa profissionalizar o serviço de vans escolares, eliminando o improviso e aumentando a segurança viária através da automação da comunicação entre pais e motoristas.


Aplicativo móvel para gestão segura de transporte escolar, com foco em:

- Motoristas de vans escolares
- Pais e responsáveis

## Stack

- Expo (SDK 55)
- React Native
- TypeScript
- `react-native-maps` para o mapa em tempo real (protótipo)

## Como rodar:

1. Instale as dependências:

```bash
npm install
```

2. Inicie o app:

```bash
npm run start
```

3. Use o app Expo Go no celular ou um emulador Android/iOS para visualizar o VansMind.

## Funcionalidades no protótipo:

- Seleção de perfil: motorista ou responsável.
- Tela de motorista:
  - Mapa em modo escuro com rota e paradas.
  - Marcador da van.
  - Lista de alunos/rotas em um painel inferior.
  - Botão grande para confirmar embarque do próximo aluno.
- Tela de responsável:
  - Mapa em modo escuro com rota e posição da van.
  - Ponto da casa do aluno.
  - Card de status do dia.
  - Ação para marcar falta hoje, ajustando a rota.

Este é um protótipo.


## 📂 Estrutura do Repositório

* **`assets/`**: Logotipos, ícones e recursos visuais.
* **`docs/apresentacao/`**: Contém o **Pitch Deck** com a estratégia de negócio e análise de mercado.
* **`docs/ux-ui/`**: Documentação de design, incluindo **Mapa de Empatia** ,link para vizualização do prototipo, **Jornada do Usuário** e **PERSONA**.
* **`prototype/`**: Protótipo navegável em HTML/CSS para validação de interface.
* **`src/`**: Pasta reservada para o desenvolvimento do código-fonte (Backend/Frontend).

## 🚀 Diferenciais do Projeto
* **Segurança:** Foco em reduzir o uso do celular pelo motorista durante o trajeto.
* **Logística:** Rota otimizada automaticamente com base nos "check-outs" dos alunos.
* **Transparência:** Localização em tempo real para os pais.
